# Herman · Sinasera

A first draft personal website for a geographer.

## Preview

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Pages

- `index.html` - homepage with life, writing, portfolio, and about sections
- `portfolio.html` - portfolio index page
- `articles/ntu-geography-dept-graduation-speech.html` - NTU Geography Department graduation speech
- `portfolio/baseball/index.html` - Washington Nationals 2019 baseball analysis dashboard

## Images

- `images/` - place article and gallery images here

## Fonts

- `assets/fonts/` - local font files used by the site
